## Installer
-   `git clone https://github.com/mirzapurnandi/bloglumen.git`
-   `cd bloglumen`
-   `composer install`
-   `cp .env-example .env`
-   `php artisan jwt:secret`
-   `php artisan migrate`
-   `php artisan db:seed`
